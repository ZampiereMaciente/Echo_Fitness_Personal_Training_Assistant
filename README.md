# 🏋️‍♂️ Physical Assessment App for Personal Trainers

A mobile application built with **Flutter** designed for **personal trainers** who want to perform, manage, and share physical assessments with their clients in a simple and professional way.

---

## 📱 About the Project

This app helps fitness professionals streamline their workflow by allowing them to:

- Collect **client personal data**
- Record **anthropometric measurements** (skinfolds and body circumferences)
- Automatically calculate:
  - **BMI (Body Mass Index)**
  - **BMR (Basal Metabolic Rate)**
- Generate a **PDF report**
- Easily share results with clients

---

## 🚀 Features

### 🧾 Client Registration
- Name
- Date of birth
- Phone number
- Email
- Evaluation dates

### 📏 Physical Assessment
- Skinfold measurements (triceps, subscapular, suprailiac, abdomen, etc.)
- Body circumferences (arms, forearms, etc.)

### 📊 Automatic Calculations
- BMI
- BMR

### 📄 Report Generation
- Export all data into a professional PDF
- Ready to send to clients
